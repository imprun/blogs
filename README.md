# imp-blogs

imprun.dev 기술 블로그 원본 저장소

## 구조

```
imp-blogs/
├── CLAUDE.md          # 블로그 작성 가이드
├── index.md           # 블로그 번호 ↔ 파일 매핑
├── draft/             # 작성 중인 초안
└── 2025/MM/DD/        # 발행된 블로그 (날짜별)
```

## 블로그 작성

1. `draft/` 폴더에 초안 작성
2. [CLAUDE.md](./CLAUDE.md) 가이드 준수
3. 완성 후 `2025/MM/DD/` 폴더로 이동
4. [index.md](./index.md)에 번호 추가

## 발행

- 블로그 URL: https://blog.imprun.dev/{번호}
- 최신 번호는 index.md 참고
